IDEA永久破解

idea
其他
Publish Date:   2020-03-07

IDEA是目前最流行也是最好用的集成开发工具，强大的代码提示和编辑功能吸引了很多开发者，Jetbrains下还有很多类似的开发工具，也很受开发者青睐。Jetbrains下的产品都是商业付费版，因此购买才可以使用，这对于普通开发者和学生党来说是笔小开销，到网上找一些激活码呢，过一段时间就会时期，时不时的得再去找新的激活码也很烦人，这里介绍的是如何将IDEA永久破解，愉快的写代码。当然笔者建议有一定经济条件的，还是去官网购买正版软件，也不是很贵，这是对别人产品的一种尊重也是认可。

Jetbrains的全家桶产品众多，本文主要介绍几款常用的软件破解方法，包括IDEA，WebStorm和GoLand。每个开发工具在使用不同的版本时，破解工具也要使用其对应的版本，而且破解工具不同，破解方式也不一样，并不通用。因此破解方式只针对具体的版本，如果版本和本文阐述的版本不一致，可能会导致破解失败。下面几个版本都是亲测可用的。

IDEA破解

破解准备

  1. IDEA版本：2018.1.6
  2. 破解工具：百度网盘: https://pan.baidu.com/s/1DRROpPD3CwpjiQ27xnIjgg 提取码: c1vv

破解方法

  1. 把下载好的破解工具JetbrainsIdesCrack-4.2-release-enc.jar添加到IDEA安装目录的 bin 目录下。

  2. 修改bin目录下的idea64.exe.vmoptions和idea.exe.vmoptions两个文件，在文件中最后一行添加如下配置

    -javaagent:D:\Program Files\JetBrains\IntelliJ IDEA 2018.1.6\bin\JetbrainsIdesCrack-4.2-release-enc.jar

    文件路径是破解jar包的路径，idea64.exe.vmoptions和idea.exe.vmoptions在同步操作系统上可能只有一个，这里是win10的系统。只有一个改一个就行。

  3. 打开IDEA工具，在 Activation Code中填写刚刚这idea64.exe.vmoptions文件中追加的配置

  4. 至此IDEA已破解完成，可进入到IDEA后，打开 Help -> About 查看是否破解成功，如破解成功可以看到到期时间是2099年

WebStorm破解

破解准备

  1. webstorm版本：2019.2.1
  2. 破解工具：百度网盘: https://pan.baidu.com/s/1iOEiZ07iFPWMZN7B4ZvRBg 提取码: aejt

破解方法

  1. 把下载好的破解工具jetbrains-agent.jar添加到IDEA安装目录的 bin 目录下。

  2. 修改bin目录下的webstorm.exe.vmoptions和webstorm.exe.vmoptions两个文件，在文件中最后一行添加如下配置

  3. 打开hosts文件，windows在 C:\Windows\System32\drivers\etc 目录下，在hosts文件中添加如下配置

    0.0.0.0 account.jetbrains.com
0.0.0.0 www.jetbrains.com

    这样就使得webstorm不能联网检查，然后重新打开webstorm，这时由于不能联网检查会弹出提示框，选择Activation code方式离线激活，使用ACTIVATION_CODE.txt 内的注册码激活

  4. 至此Webstorm已破解完成，可进入到Webstorm后，打开 Help -> About 查看是否破解成功，如破解成功可以看到到期时间是2089年
