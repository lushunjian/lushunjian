彗星来的那一夜 / Coherence
2020-02-23 / 科幻 太空 宇宙 / ★★★★★ 力荐
这是一部关于平行宇宙的电影，小成本硬科幻。虽然造成时刻混乱的原因有点不靠谱，但是本片的重点是平行宇宙时空错乱后发生的各种诡异事件。本片前面一部分会有点乏味，不过喜欢推理的一定不要错过，这部分有很多细节，都在暗示时空错乱从影片一开始就存在了。
源代码 / Source Code
2020-02-23 / 科幻 动作 / ★★★★★ 力荐
首先说《源代码》的理论： 1、人的大脑可以记录死亡前8分钟的场景； 2、灯关闭后有余晖，人死亡后大脑也能保持8分钟的回路，这个回路类似于容器，可装入其他记忆、可编程，这就是源代码区。 整个电影围绕这个理论展开，电影最后男主似乎创造了一个新的平行宇宙。这是我看过的最好的关于平行宇宙的电影，剧情很精彩
银翼杀手2049 / Blade Runner 2049
2020-02-19 / 人工智能 经典 科幻 2017 反乌托邦 / ★★★★★ 力荐
一部经典的科幻片续集，讲述仿生人获得自我意识，想要被人类认可的故事。机器人也有直觉。它们不只是可以飞快运算的机器，它们也会思考、睡觉和做梦。由此引发了一个更大的问题：什么才是人的特质？
攻壳机动队 / 攻殻機動隊
